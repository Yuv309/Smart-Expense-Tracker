# Smart-Expense-Tracker
A Flask-based web app to track daily expenses

## 📌 About
This is a web application made using Flask and SQLite to manage daily expenses.

## 🚀 Features
- Add expenses
- View records
- Simple interface

## 🛠 Technology
- Python
- Flask
- SQLite

## ▶️ How to Run
1. Install Flask
2. Run app.py
3. Open browser at 127.0.0.1:5000
